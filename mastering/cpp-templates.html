Mastering C++ Templates

Explore the power of templates in C++ and how they enable generic programming.

What are C++ Templates?

C++ templates allow you to write generic and reusable code. They enable you to define functions and classes that work with any data type, making your code more flexible and maintainable.

Key Concepts

  • Function Templates: Write generic functions that work with any data type.
  • Class Templates: Create generic classes that can operate on different data types.
  • Template Specialization: Customize templates for specific data types.

Example Code

#include <iostream>
using namespace std;

// Function Template
template <typename T>
T add(T a, T b) {
    return a + b;
}

// Class Template
template <typename T>
class Box {
private:
    T value;
public:
    void setValue(T v) {
        value = v;
    }
    T getValue() {
        return value;
    }
};

int main() {
    // Function Template Example
    cout << "Sum (int): " << add(5, 10) << endl;
    cout << "Sum (double): " << add(3.5, 2.7) << endl;

    // Class Template Example
    Box<int> intBox;
    intBox.setValue(123);
    cout << "Box Value: " << intBox.getValue() << endl;

    return 0;
}
