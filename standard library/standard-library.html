C++ Standard Library

Discover the C++ Standard Library, including containers, algorithms, and utilities.

What is the C++ Standard Library?

The C++ Standard Library is a collection of classes and functions that provide common programming tasks like data structures, algorithms, and input/output operations. It is part of the C++ Standard Template Library (STL).

Key Components

  • Containers: Data structures like vectors, lists, maps, and sets.
  • Algorithms: Functions for sorting, searching, and manipulating data.
  • Utilities: Tools like smart pointers, tuples, and function objects.

Example Code

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {5, 2, 9, 1, 5, 6};
    std::sort(numbers.begin(), numbers.end());

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}
